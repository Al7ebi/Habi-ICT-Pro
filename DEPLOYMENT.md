# 🚀 دليل النشر على Streamlit Cloud\n\n## الخطوة 1️⃣: تحضير المشروع محليًا\n\n### تأكد من أن لديك:\n- ✅ كل الملفات موجودة\n- ✅ `requirements.txt` به كل المكتبات\n- ✅ `.env.example` بدلاً من `.env` (لا تضف API Keys)\n- ✅ `.gitignore` يحتوي على `.env`\n\n### الملفات الأساسية المطلوبة:\n```\nHABIICT_PRO/\n├── app.py                    ← الملف الرئيسي\n├── config.py\n├── requirements.txt\n├── README.md\n├── .gitignore\n├── .streamlit/config.toml\n├── core/\n│   ├── __init__.py\n│   ├── data_fetcher.py\n│   └── ict_logic.py\n└── agents/\n    ├── __init__.py\n    ├── director.py\n    ├── quant_ict.py\n    └── risk_manager.py\n```\n\n## الخطوة 2️⃣: رفع المشروع على GitHub\n\n### أ) إنشاء Repository جديد على GitHub:\n\n1. ادخل إلى https://github.com/new\n2. اسم الـ Repo: `HABIICT_PRO` (أو أي اسم تريده)\n3. الوصف: \"Smart Money Concepts Analyzer - ICT Trading Tool\"\n4. Private أو Public (اختر حسب الحاجة)\n5. اضغط \"Create Repository\"\n\n### ب) رفع الملفات:\n\n```bash\n# في مجلد المشروع\ngit init\ngit add .\ngit commit -m \"Initial commit: HABIICT PRO v1.0\"\ngit branch -M main\ngit remote add origin https://github.com/YOUR_USERNAME/HABIICT_PRO.git\ngit push -u origin main\n```\n\n## الخطوة 3️⃣: النشر على Streamlit Cloud\n\n### أ) التسجيل على Streamlit Cloud:\n\n1. ادخل إلى https://streamlit.io/cloud\n2. اضغط \"Sign up\"\n3. اختر GitHub (لتوصيل حسابك تلقائياً)\n4. وافق على الصلاحيات\n\n### ب) إنشاء تطبيق جديد:\n\n1. اضغط \"New app\"\n2. ملء الحقول:\n   - **Repository**: `YOUR_USERNAME/HABIICT_PRO`\n   - **Branch**: `main`\n   - **Main file path**: `HABIICT_PRO/app.py` ← مهم!\n3. اضغط \"Deploy\"\n\n### ج) انتظر النشر (2-5 دقائق):\n\nستظهر الرسالة:\n```\n✅ App deployed successfully!\nYou can view it at: https://username-habiict-pro.streamlit.app\n```\n\n## الخطوة 4️⃣: إضافة Secrets (API Keys)\n\n### ⚠️ هام: لا تضع API Keys في الكود!\n\n1. ادخل إلى لوحة تحكم التطبيق\n2. اضغط على الثلاث نقاط ⋮ (في الأعلى يميناً)\n3. اختر \"Settings\"\n4. اختر \"Secrets\"\n5. أضف الـ secrets بصيغة TOML:\n\n```toml\nBINANCE_API_KEY = \"your_actual_api_key_here\"\nBINANCE_API_SECRET = \"your_actual_secret_here\"\nTRADINGVIEW_API = \"your_trading_view_api_here\"\n```\n\n6. اضغط Save\n\n## الخطوة 5️⃣: التحديثات المستقبلية\n\nكلما تحديث الكود:\n\n```bash\ngit add .\ngit commit -m \"Update: description of changes\"\ngit push\n```\n\nStreamlit Cloud سيقوم بـ redeploy تلقائياً!\n\n## 🔗 URL التطبيق النهائي:\n\n```\nhttps://YOUR_USERNAME-habiict-pro.streamlit.app\n```\n\nمثال:\n```\nhttps://mohammedali-habiict-pro.streamlit.app\n```\n\n## ⚙️ إعدادات إضافية (اختيارية)\n\n### تغيير اسم التطبيق:\n\n1. في لوحة التحكم\n2. اضغط على Settings ⚙️\n3. اضغط على اسم التطبيق\n4. غيّر الاسم\n\n### فعّل التحديثات التلقائية:\n\n1. Settings ⚙️\n2. اختر \"Rerun script on update\"\n3. اختر التحديثات الدورية\n\n## 🐛 حل المشاكل الشائعة\n\n### المشكلة: \"Main file path not found\"\n\n**الحل:**\n- تأكد من أن `app.py` موجود في المجلد الصحيح\n- المسار يجب أن يكون: `HABIICT_PRO/app.py`\n\n### المشكلة: \"ModuleNotFoundError\"\n\n**الحل:**\n- تأكد من `requirements.txt` يحتوي على كل المكتبات\n- جرّب تشغيل محليًا: `streamlit run app.py`\n\n### المشكلة: \"API Key not found\"\n\n**الحل:**\n- تأكد من إضافة Secrets بشكل صحيح\n- أعد تسمية المتغيرات إذا لزم الأمر\n\n### المشكلة: البيانات لا تحمّل\n\n**الحل:**\n- Binance API قد يكون معطل\n- جرّب Symbol مختلفة\n- تحقق من الـ internet connection\n\n## ✅ التحقق من النشر:\n\n1. ادخل إلى URL التطبيق\n2. جرّب كل الوظائف:\n   - [ ] اختر عملة\n   - [ ] اختر timeframe\n   - [ ] جرّب التحليل\n   - [ ] شاهد الرسم البياني\n   - [ ] عدّل إعدادات المخاطر\n\n## 📞 الدعم:\n\nإذا واجهت مشاكل:\n\n1. تحقق من Streamlit Logs:\n   - Settings → View logs\n\n2. اسأل على Streamlit Community:\n   - https://discuss.streamlit.io\n\n3. تحقق من GitHub Issues:\n   - https://github.com/streamlit/streamlit/issues\n\n---\n\n**مبروك! 🎉 تطبيقك الآن حي على الإنترنت!**\n\nشارك الـ URL مع الآخرين:\n```\nhttps://YOUR_USERNAME-habiict-pro.streamlit.app\n```\n"
